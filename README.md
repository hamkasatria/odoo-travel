# odoo-travel

Dummy travel system for having fun :)

- framework : `Golang Fiber`
- database : `mongodb`
### How to Run
1. install prerequirment `golang` and `mongodb`
2. set up environtment with command `cp .env.example .env`
3. run this server with command `go run main.go`
